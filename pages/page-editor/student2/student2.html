2. Technical Description:

Table Page

The Table Page utilized semantic HTML elements like <table>, <thead>, <tbody>, and <tr> to structure data effectively. Styling with CSS properties like border-collapse, padding, and text-align was applied for a clean and responsive table layout.

Navbar & Template

The Navbar was dynamically rendered using an <iframe> tag, allowing for consistent navigation across different pages. The template was created to ensure that the navigation bar is reusable and can be updated in one place, without needing to modify each page individually.
